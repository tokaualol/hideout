## Super Mario 64
<hr>

[Ukikipedia](https://ukikipedia.net)<br>
Extensive wiki on Super Mario 64 Glitches, Courses, Guides, Game Mechanics, and other sorts of information.

[HackerSM64](https://github.com/HackerN64/HackerSM64)<br>
A modified version of the SM64 decompilation project that includes numerous enhancements, bugfixes, and quality-of-life improvements for ROM hackers.

[Parallel Launcher](https://parallel-launcher.ca)<br>
A launcher for the ParallelN64 emulator, with romhacking.com integration.

[Bowser's Blueprints](https://blueprint64.ca)<br>
A general-purpose ROMhacking tool for creating Super Mario 64 ROMhacks.

[Fast64](https://github.com/Fast-64/fast64)<br>
Blender based editor for scenes, meshes, and more in first-party N64 games.

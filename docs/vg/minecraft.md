## Minecraft
Minecraft related tools or other software.
<hr>

[MCTools](https://mc-tools.net/)<br>
A website containing many easy-to-use tools for creating, editing and converting things for Minecraft. 

[MCA Selector](https://github.com/Querz/mcaselector)<br>
A tool to export or delete selected chunks and regions from a Java Edition world.

[BDEngine](https://bdengine.app)<br>
Create and animate models for Minecraft.

[Block Display Models](https://block-display.com)<br>
Catalog for Minecraft models.

[Birdflop](https://birdflop.com/resources)<br>
Useful tools for developers.

[MC Utils](https://mcutils.com)<br>
Community powered Minecraft utilities for developers, builders and players.

[MC Assets](https://mcasset.cloud)<br>
Browse Minecraft assets, including Textures, Sounds, Models, Fonts, Shaders, and more.

[Cubical](https://cubical.xyz)<br>
Online editor for Minecraft builds.

[Mineways](https://realtimerendering.com/erich/minecraft/public/mineways)<br>
Turn your builds into 3D models you can use for making images, movies, or 3D prints.

[ZenithProxy](https://github.com/rfresh2/ZenithProxy)<br>
Minecraft bot proxy focused on 2b2t, but has a ton of other amazing featues, chat relay, replaymod, spectator mode.

[SniffCraft](https://github.com/adepierre/SniffCraft)<br>
C++ Minecraft proxy that logs all packets.

[MapartCraft](https://rebane2001.com/mapartcraft)<br>
A Minecraft mapart schematic and map.dat generator.

[Modrinth](https://modrinth.com)<br>
The place for Minecraft mods, resource packs, data packs, shaders, modpacks, plugins, and servers!

## Launcher(s)

[Prism Launcher](https://prismlauncher.org)<br>
An open-source Minecraft launcher with the ability to manage multiple instances, accounts and mods.

[Pandora](https://pandora.moulberry.com)<br>
A modern Minecraft launcher that balances ease-of-use with powerful instance management features.

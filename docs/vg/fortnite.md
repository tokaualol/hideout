## Fortnite
[FortniteEndpointsDocumentation](https://github.com/LeleDerGrasshalmi/FortniteEndpointsDocumentation)<br>
Unofficial API Documentation for official Fortnite & Epic Games services.

[Project Reboot 3.0](https://github.com/Milxnor/Project-Reboot-3.0)<br>
Project Reboot is an open source game server for Fortnite, featuring support for versions between Season 3 and Season 15, and partial support for other versions.

[Erbium](https://github.com/plooshi/Erbium)<br>
Erbium is an open source WIP universal game server for Fortnite, featuring support for versions 3.40 to 19.40, with partial support for Chapter 3+ seasons.

[Nova](https://github.com/projectnovafn/Nova)<br>
Nova was the core game server for Project Nova, a Fortnite private server developed between 2022 and 2025.

[Nova Matchmaker](https://github.com/projectnovafn/NovaServices.Matchmaker)<br>
The matchmaker for Project Nova, a Fortnite private server developed between 2022 and 2025.

[Nova Client](https://github.com/projectnovafn/NovaLauncher.Client)<br>
The client DLL for Project Nova, a Fortnite private server developed between 2022 and 2025.

[Reboot Launcher](https://github.com/Auties00/Reboot-Launcher)<br>
Reboot Launcher is a GUI and CLI Launcher made for the Reboot Project, but can be used with any valid client and server DLLs.

[LawinServerV1](https://github.com/Lawin0129/LawinServer)<br>
LawinServer is a local singleplayer lobby server emulator for Fortnite, supporting Battle Royale and Save the World.

[LawinServerV2](https://github.com/Lawin0129/LawinServerV2)<br>
LawinServer V2 is a fortnite backend written in Node.js that features an account system and XMPP for certain seasons. Only supports Battle Royale.

[Cobalt](https://github.com/Milxnor/Cobalt)<br>
Cobalt is a Universal SSL Bypass for Fortnite.

[Tellurium](https://github.com/plooshi/Tellurium)<br>
Tellurium is a universal redirect/SSL bypass for native UE HTTP games. Originally designed for Fortnite.

[Sinum](https://github.com/projectnovafn/Sinum)<br>
Sinum is used to redirect requests from a target URL, such as ol.epicgames.com, to another server, like api.novafn.dev. Supports Android, iOS, Windows, and Xbox.
